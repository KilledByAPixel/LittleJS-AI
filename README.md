A collection of games made using [LittleJS](https://github.com/KilledByAPixel/LittleJS) and AI-assisted vibe coding experiments. All of these games were built completely by AI and can be used as starting points to remix into new games.

This repo also contains tools and starter projects to help with AI agentic development.

You can use the [LittleJS GPT AI](https://chatgpt.com/g/g-67c7c080b5bc81919736bc8815836be6-make-games-with-littlejs) to make your own games.

## About

This repository showcases games built with the lightweight JavaScript game engine **LittleJS** combined with AI coding assistants. It's an exploration of how AI can help rapidly prototype and develop retro-style games.

[LittleJS](https://killedbyapixel.github.io/LittleJS/) is a super lightweight 2D JavaScript game engine with fast WebGL rendering. It's perfect for game jams, prototypes, and learning game development.
